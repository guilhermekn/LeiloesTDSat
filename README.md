# 🏷️ LeiloesTDSat

> Sistema de leilões desenvolvido como projeto prático.

---

## 📋 Sobre o Projeto

O **LeiloesTDSat** é uma aplicação voltada para o gerenciamento de leilões, permitindo o cadastro, listagem e controle de itens e lances de forma organizada e eficiente.

---

## 🚀 Tecnologias Utilizadas

- ☕ **Java** — Linguagem principal
- 🖥️ **NetBeans** — IDE de desenvolvimento
- 🐙 **Git & GitHub** — Controle de versão

> ⚠️ *Atualize esta seção com as tecnologias reais do seu projeto.*

---

## ⚙️ Como Executar

```bash
# Clone o repositório
git clone https://github.com/guilhermekn/LeiloesTDSat.git

# Abra no NetBeans
# File > Open Project > selecione a pasta clonada

# Execute o projeto
# Clique em Run > Run Project (F6)
```
